## 📝 Project Overview  
This project showcases an **interactive Sales Performance Dashboard** created in Microsoft Excel to monitor and analyze sales executives’ performance across different cities.  

The dashboard provides insights into total sales, target achievement percentage, and performance gaps, helping businesses evaluate employee productivity and improve sales strategies.

---

## 🎯 Purpose  
- Track sales executive performance across multiple cities  
- Compare total sales and target achievement rates  
- Identify employees performing below target  
- Enable data-driven sales management decisions  
- Provide an interactive and easy-to-use reporting system  

---

## 🛠️ Tech Stack  
- 📗 **Microsoft Excel** – Dashboard creation & data analysis  
- 📊 **Pivot Tables** – Data summarization  
- 📈 **Charts & Graphs** – Data visualization  
- 🎛️ **Form Controls / Interactive Buttons** – Dashboard navigation  
- 📐 **Conditional Formatting** – Highlighting KPIs and trends  

---

## 📂 Data Source  
- Sales dataset containing:  
  - Sales executive names  
  - City-wise sales records  
  - Total sales values  
  - Target achievement percentages  
  - Away-from-target performance metrics  

- Dashboard created as part of hands-on Excel analytics practice.

# 🚀 Features & Highlights  

## 🔍 Business Problem  
Organizations often struggle to monitor sales executive performance across different regions and identify employees who are underperforming or missing targets.

Without proper visualization and analysis, it becomes difficult to:
- Track sales progress effectively  
- Compare regional performance  
- Identify gaps in target achievement  
- Make informed business decisions  

---

## 🎯 Goal  
To design an **interactive Excel dashboard** that provides a clear overview of:
- Sales performance by executive  
- City-wise sales comparison  
- Target achievement analysis  
- Performance gaps and trends  

---

## 📈 Key Dashboard Metrics  
- 📍 City-wise Sales Analysis  
- 📊 Total Sales by Sales Executive  
- 🎯 Target Hit Percentage  
- 📉 Away From Target Percentage  
- 🧑‍💼 Employee Performance Comparison  

---

## 📊 Key Visuals  
- 📌 Interactive City Selection Buttons  
- 📋 Executive-wise Sales Tables  
- 📊 Bar Chart – Total Sales Comparison  
- 🥧 Pie Chart – Target Achievement Distribution  
- 📉 Line Chart – Away From Target % Analysis  
- 🎛️ Interactive Dashboard Navigation Controls  

---

## ⭐ Key Features  
- Interactive dashboard navigation using buttons  
- Dynamic city-wise filtering  
- Comparative employee performance analysis  
- Easy-to-understand visual reports  
- Professional and structured dashboard design  

---

## 💡 Key Insights  
- 🔹 Certain sales executives consistently outperform others in total sales  
- 🔹 Some employees show high “Away From Target %”, indicating improvement areas  
- 🔹 City-wise comparison helps identify stronger sales regions  
- 🔹 Visual performance tracking simplifies decision-making for managers
